# NestJS REST API

A CRUD REST API using the NestJS framework and MongoDB/Mongoose.

## Setup

Add your mongodb uri to the "config/keys.ts file"

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
